---
description: Run the Document Extraction & Chat Application
---

1. **Start the Backend Server**
   Open a terminal in the project root (`task2`) and run:
   ```powershell
   py api.py
   ```
   *Wait for "Server is starting" message.*

2. **Start the Flutter Frontend**
   Open a **new** terminal, navigate to the frontend folder, and run the app:
   ```powershell
   cd frontend_flutter
   flutter run -d windows
   ```
   *(You can change `-d windows` to `-d chrome` or another device if preferred)*
